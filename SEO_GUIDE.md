# SEO Setup Guide

## Google Analytics Kurulumu

1. [Google Analytics](https://analytics.google.com/) hesabı oluştur
2. Yeni property oluştur: `semihaydin.dev`
3. Measurement ID'yi al (G-XXXXXXXXXX formatında)
4. `public/index.html` dosyasındaki `G-Y7W3CTSGT6` kodunu kendi kodunla değiştir

## Google Search Console Kurulumu

1. [Google Search Console](https://search.google.com/search-console) aç
2. "Add Property" > "URL prefix" seç
3. `https://semihaydin.dev` ekle
4. Doğrulama için HTML tag metodunu seç
5. Verilen meta tag'i `public/index.html` içine ekle

## Sitemap Gönderimi

1. Search Console'da "Sitemaps" bölümüne git
2. `https://semihaydin.dev/sitemap.xml` URL'ini ekle
3. "Submit" butonuna tıkla

## robots.txt Kontrolü

Site yayına alındıktan sonra kontrol et:
- `https://semihaydin.dev/robots.txt` erişilebilir olmalı
